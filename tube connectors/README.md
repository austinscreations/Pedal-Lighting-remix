# Tube Connectors
The main reason I wanted magnetic connectors was for the stacking support. The also allow the tubes to assemble fairly easily. But any tube that plugs into a base can be switched and easily used in a stacking configuration. all you have to do is slide a connection plate in place (held in a with two thumb screws) and you have a rather resilent stacking tube.
