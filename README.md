# Pedal Lighting Remix
This work is based on the work done by [Pedal Lighting team](https://www.petallighting.com/)

I wanted to use magnetic connectors so that any tube could be used in a stacked configuration. Then while designing the cost of the laser cut base lead me to design one that could be 3d printed.

Because the orginal project is opensource I've openned these files in the same spirit for others to use / adapt for their rigs
